## Key Features

<FeatureGrid>
  <Feature icon="Play" title="Muted autoplay">
    Plays automatically (muted) for inline previews
  </Feature>
  <Feature icon="Image" title="Poster image">
    Display a thumbnail before video plays
  </Feature>
  <Feature icon="Video" title="Native controls">
    Full browser video controls for playback
  </Feature>
  <Feature icon="AspectRatio" title="Aspect ratio control">
    Support for auto, 1:1, 4:3, 16:9, and 9:16 formats
  </Feature>
</FeatureGrid>

## Source and Install

Copy [`components/tool-ui/video`](https://github.com/assistant-ui/tool-ui/tree/main/components/tool-ui/video) and the [`shared`](https://github.com/assistant-ui/tool-ui/tree/main/components/tool-ui/shared) directory into your project.

### Prerequisites

```bash
pnpm dlx shadcn@latest add badge button card tooltip
```

### Download

- [**Shared Dependencies (GitHub)**](https://github.com/assistant-ui/tool-ui/tree/main/components/tool-ui/shared)
- [**Video (GitHub)**](https://github.com/assistant-ui/tool-ui/tree/main/components/tool-ui/video)
