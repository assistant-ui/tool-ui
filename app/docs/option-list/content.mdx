## When to Use Option List

- Inline confirmation for multi-select tool outputs
- Single-choice pickers without modal flows
- Quick "add to cart" or filter selection in chat results

## Key Features

- **Dual modes**: checkbox-style multi or radio-style single select
- **Async-friendly**: confirm button shows a spinner while awaiting your handler
- **Constraints**: `minSelections`/`maxSelections` and alignment/layout tweaks
- **Serializable**: Zod schemas ship alongside (`OptionListPropsSchema`)

## Install

Copy `components/tool-ui/option-list` into your project and ensure the shadcn-style button and separator atoms are available (or adjust `_ui.tsx` to your paths).
