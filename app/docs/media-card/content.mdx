An information surface for links, images, video, and audio—optimized for inline previews in chat.

Use Media Card when a tool returns a **single piece of content** you want to preview, highlight, and optionally act on.

## When to use Media Card

Good fits:

- Link previews (articles, docs, dashboards).  
- Image / video / audio thumbnails.  
- File / document summaries.

Avoid Media Card for:

- Large galleries (use multiple cards or a Data Table instead).  
- Long-form content editing (use a dedicated surface).

## Rendering tool outputs

### Server – define a preview tool

```ts
import { streamText, convertToModelMessages, tool, type UIMessage } from 'ai';
import { openai } from '@ai-sdk/openai';
import { z } from 'zod';
import { serializableMediaCardSchema } from '@/components/media-card/schema';

export async function POST(req: Request) {
  const { messages }: { messages: UIMessage[] } = await req.json();

  const result = streamText({
    model: openai('gpt-4o'),
    messages: convertToModelMessages(messages),
    tools: {
      previewLink: tool({
        description: 'Show a preview card for a URL',
        inputSchema: z.object({ url: z.string().url() }),
        outputSchema: serializableMediaCardSchema,
        async execute({ url }) {
          // Replace this stub with real metadata fetching
          return {
            id: 'link-1',
            kind: 'link',
            href: url,
            title: 'Example site',
            description: 'Preview of the requested URL',
            thumb:
              'https://images.unsplash.com/photo-1633356122544-f134324a6cee?auto=format&fit=crop&q=80&w=1200',
            meta: { source: new URL(url).hostname },
          };
        },
      }),
    },
  });

  return result.toUIMessageStreamResponse();
}
```

### Client – parse and render

```tsx
import { useChat } from '@ai-sdk/react';
import { DefaultChatTransport } from 'ai';
import { MediaCard, parseSerializableMediaCard } from '@/components/media-card';

export default function Chat() {
  const { messages } = useChat({
    transport: new DefaultChatTransport({ api: '/api/chat' }),
  });

  return (
    <div>
      {messages.map((m) => (
        <div key={m.id}>
          {m.parts.map((part, i) => {
            if (part.type === 'tool-previewLink' && part.state === 'output-available') {
              const cardProps = parseSerializableMediaCard(part.output);
              return (
                <MediaCard
                  key={i}
                  {...cardProps}
                  maxWidth="420px"
                  actions={[
                    { id: 'open', label: 'Open' },
                    { id: 'save', label: 'Save' },
                  ]}
                  onAction={(id, card) => {
                    if (id === 'open') {
                      window.open(card.href, '_blank');
                    }
                    if (id === 'save') {
                      // Your app logic here
                    }
                  }}
                />
              );
            }
            return null;
          })}
        </div>
      ))}
    </div>
  );
}
```
