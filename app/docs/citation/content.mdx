## Key Features

<FeatureGrid>
  <Feature icon="FileText" title="Source attribution">
    Display title, domain, author, and publication date
  </Feature>
  <Feature icon="Quote" title="Snippet preview">
    Show relevant excerpts from the source
  </Feature>
  <Feature icon="Layers" title="Three variants">
    Default card, inline chips, or stacked favicons
  </Feature>
  <Feature icon="Link" title="Click to navigate">
    Open source in new tab with proper security
  </Feature>
</FeatureGrid>

## When to Use

- **Good for:** RAG citations, research references, documentation sources, API documentation links
- **Not for:** Link previews with OG images (use [Link Preview](/docs/link-preview)), generic URL cards

## Source and Install

Copy [`components/tool-ui/citation`](https://github.com/assistant-ui/tool-ui/tree/main/components/tool-ui/citation) and the [`shared`](https://github.com/assistant-ui/tool-ui/tree/main/components/tool-ui/shared) directory into your project.

### Prerequisites

```bash
pnpm dlx shadcn@latest add button tooltip popover
```
