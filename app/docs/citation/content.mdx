## When to Use

- **Good for:** RAG citations, research references, documentation sources, API documentation links
- **Not for:** Link previews with OG images (use [Link Preview](/docs/link-preview)), generic URL cards

## Source Types

The `type` prop customizes the icon displayed:

- `webpage` — General web pages (default)
- `document` — Documentation, guides, manuals
- `article` — News, blog posts, research papers
- `api` — API documentation, endpoints
- `code` — Code repositories, snippets
- `other` — Miscellaneous sources

## Source and Install
